学习前端

  • 什么是前端？

    说到前端我们就要先了解网页开发的三剑客，那是哪三剑客呢，其实啊也就是HTML，CSS，JavaScrip，那什么是HTML，CSS和JavaScrip呢

    • 什么是HTML
      • 其实啊HTML是一种超文本标记语言，并且啊他是现在万维网的基础
      • HTML他是通过标签来描述网页中的内容的
    • 什么是CSS

      其实啊这个也很简单，其实啊CSS就是一种声明式语言，但是啊这里要注意的就是他不是编程语言，因为他没有逻辑，并且啊CSS他的主要的作用就是用来美化我们的网页的

    • 什么是Javascript

    并且啊理解前端这个概念的话其实也很简单，其实啊前端就是我们用户能看见的界面，就是用前端的代码来写的，也就是用前端三剑客来写的，了解完前端这个概念之后呢，我们就可以正式的开始学习前端了

  • 首先的话先学习HTML语言
    • 在学习HTML之前我们必须要先了解三个概念之后呢才能开始学，那是哪三个概念呢，其实啊也就是标签，元素，属性
      • 什么是标签？
        • 其实啊标签这个概念也很简单，标签其实就是在HTML中最基础的东西，并且啊在HTML中标签的主要的作用就是告诉浏览器，这个标签中的内容是什么意思，也就是告诉浏览器用什么方式来解析里面的内容
      • 什么是元素？
        • 元素其实就是开始标签到结束标签的这个整体就称之为元素
      • 什么是属性？
        • 其实啊属性就是这个标签的额外信息，并且啊属性他是不会显示出来的，他只会影响这个标签的最终显示出来的效果
    • 之后呢了解完这些HTML的基础概念我们就可以正式的开始学习HTML语言了，首先我们先学习一下HTML网页的基本结构，如下
    <!DOCTYPE html>
<html>
	<head>
		<meta charset = "utf-8"/>
		<title></title>
	</head>
	<body>
	<!-- 演示 -->
	</body>
</html>
    • 如上就这个网页的基本结构
      • 首先是<!DOCTYPE html>这句代码是什么意思呢，其实啊这句代码的意思就是声明这个文档的类型为html
      • 然后就是<html></html>这个标签，我们可以看到这个标签他是一个双标签，也就是说如果一个标签他有结束标签的话，那么这个标签就是双标签，否则就是单标签，然后就是这个html标签，这个标签他是这个网页的根标签，也就是说除了<!DOCTYPE html>这个，其他的标签都要写在这个根标签的内部
      • 然后的话就是head这个标签，这个标签的话他是头部标签，并且啊头部标签有一个特点就是写在头部里面的标签是不会显示在网页中的
      • 然后就是这个标签，也就是meta这个标签，这个标签的主要的作用就是设置这个网页的一些元信息，然后就是这个标签的charset这个属性，这个属性的话他的主要的作用就是用来设置浏览器是以什么样的编码来解析这个文件的
      • 然后的话就是这个标签，也就是title这个标签，这个标签的话他的主要的作用就是设置这个网页的头部标签的，也就是在标签页上显示的标题
      • 然后就是这个标签，也就是body，那这个是什么标签呢，其实啊这个标签就是这个网页的主体标签，并且这个标签啊他还有一个特点，也就是写在他内部的标签他都会显示出来，这就是head标签和body标签之间重要的区别
      • 然后的话就是这个标签，也就是<!-- 演示 -->那这个是什么呢，其实啊这个就是在html语言中的注释
    • 这就是在html网页中的基本结构了
    • 我们了解完了html网页的基本结构了之后呢，我们就可以开始学习在html中各种各样的标签了
      • <html></html> 那这个是什么标签呢，其实啊这个标签是一个根标签，并且啊除了<!DOCTYPE html>这个其他的标签都必须要写到这个标签的里面
      • <meta/> 那这个是什么标签呢，其实啊这个标签的作用就是用来这个网页的一些元信息的，就比如说设置这个网页解析的方式，也就是设置这个网页的编码，这个的话就是通过这个标签的charset这个属性来设置的
      • <title></title> 那这个是什么标签呢，其实啊这个标签是一个用来给网页设置头部标题的标签
      • <head></head> 那这个是什么标签呢，其实啊这个标签是头部标签，并且啊在头部标签里面写的标签都不会显示出来
      • <body></body> 那这个是什么标签呢，其实啊这个标签是主体标签，并且啊写到主体标签里面的标签都会被显示出来
      • <hn></hn>(m的范围在：1-6) 那这个是什么标签呢，其实啊这个标签是标题标签，并且啊标签等级是1的标题标题标签的文本是最大的，而标签等级是6的标题标签的文本是最小的
      • <p></p> 那这个是什么标签呢，其实啊这个标签是段落标签
      • <br/>那这个是什么标签呢，其实啊这个标签是换行标签，这个标签的话他的主要的作用就是给其他标签或者是文本换行
      • <hr/> 那这个是什么标签呢，其实啊这个标签是分割线标签，这个标签的话他的主要的作用就是会创建一个分割线
      • <a></a> 那这个是什么标签呢，其实啊这个标签是一个超链接标签，这个标签的话他的主要的作用就是，只要是写到这个标签里面的其他标签，只要是点击到写到超链接里面的标签的话他就会跳转我们通过href这个属性设置的网站或者是网页，这个标签的格式是这样的：<a href = “” target = “”></a>，注意我们已经知道了href属性是用来设置跳转的网页或者是网站的地址的并且啊他也可以链接其他资源但是呢如果一些资源无法展示出来的话他就会一下载的方式来给用户进行展示，并且啊这个标签的话他还有一个的作用就是他可以跳转锚点，也就是这样：<a href = “#t”></a>，并且啊必须要在你创建的这个锚点的名字前面加上井号，那如何创建锚点呢，创建锚点的话其实也很简单，也就是通过其他的标签的id这个属性就可以来进行创建锚点了
      • <img src = “” alt = “” width = “” height = “”/> 那这个是什么标签呢，其实啊这个标签是图片标签，这个标签的话他的主要的作用向网页中插入标签的，我们可以看到这个标签的话他又很多属性，首先是第一个属性，也就是src这个属性，这个属性的话他的主要的作用就是设置这个图片的路径的，然后就是alt这个属性，这个属性的话他的主要的作用就是当这个图片资源加载失败的时候就会显示这个文本来提示用户的，然后就是width这个属性，这个属性的话他的作用就是设置这个图片的宽度的，并且啊如果没有设置这个图片的高度的话他就会跟随的宽度进行自适应，也就是他会根据设置的宽度等比例的缩放高度，height这个属性也是一样的道理
      • <em></em> 那这个是什么标签呢，其实啊这个标签的作用就是用于给文本添加斜体效果的
      • <strong></strong> 那这个是什么标签呢，其实啊这个标签的作用就是用于给文本添加粗体效果的
      • <ol> <li> </li> </ol> 那这个是什么标签呢，其实啊这个标签是有序列表标签，其中啊<li></li>这个标签啊的作用就是向这个有序列表中添加内容的
      • <ul type = “”> <li> </li> </ul> 那这个是什么标签呢，其实啊这个标签是跟那个有序列表标签是差不多的，只不过列表的前面是项目符号，然后就是type这个属性，那这个是什么属性呢，其实啊这个属性就是用来设置这个列表前面的项目符号的样式的，那这个属性有多少种属性值呢，首先是第一种属性值，也就是square，设置了这个属性值之后呢列表前面的项目符号就会变成一个实心的方块，然后就是第二中属性值，也就是circle，设置了这个属性值之后呢前面的项目符号就会变成一个空心的圆点，然后就是第三种属性值，也就是disc，设置了这个属性值之后呢前面的项目符号就变成一个实心的圆点，并且啊默认也是这个，然后就是第四种属性值，也就是none，设置了这个属性值之后呢列表前面就没有这个项目符号了
      • <table width = “” height = “” align = “” bgcolor = ““ cellpadding = “” ></table> 那这个是什么标签呢，其实啊这个标签是表格标签，这个标签的主要的作用就是创建一个表格，然后就是这个表格标签的一些属性，首先是第一个属性，也就是width这个属性，这个属性的话他的主要的作用就是设置这个表格的宽度，然后就是第二个属性，也就是height这个属性，这个属性的话他的主要的作用就是设置这个表格的高度的，然后就是第三个属性，也就是bgcolor这个属性，这个属性的话他的主要的作用就是设置这个表格的背景颜色的，然后就是第四个属性，也就是cellspacing这个属性，这个属性的话他的主要的作用就是设置这个单元格和单元格之间的间距的，然后就是第五个属性，也就是cellpadding这个属性，这个属性的话他的主要的作用就是设置这个单元格里面的内容和单元格之间的间距的
      • <tr height = “” bgcolor = “” align = “” valign = “”></tr> 那这个是什么标签呢，其实啊这个标签的话他的主要的作用就是向表格中添加单元行的，并且啊这个并且啊这个标签必须要配合这个表格标签一起使用，然后的话就是这个标签的一些属性，因为啊前面的两个属性我们都已经了解过了，所以我就从第三个属性开始，也就是align这个属性，这个属性的话他的作用就是设置这个单元行中的单元格中的内容夫人对齐方式的而第四个属性他是设置在垂直方向上的对齐方式的
      • <td width = “” height = “” bgcolor = “” align = “” rowspan = “” colsspan = “”></td> 那这个是什么标签呢，其实啊这个标签的话他的作用就是向这个单元行中添加单元格的，然后就是这个标签的一些属性，因为前面的4个属性的话我们都已经了解过了，所以我们就从第五个属性开始，也就是rowspan这个属性，这个属性的话他的主要的作用就是向下创建一个单元格并合并而第六个属性的话他是向左创建一个单元格并合并
      • <th width = “” height = “” bgcolor = “” align = “” rowspan = “” colsspan = “”></th> 那这个是什么标签呢，其实啊这个标签和这个<tr></tr>是一模一样的，只不过写到这个标签里面的文本会被自动加粗
      • <textarea row = “” cols = “”></textarea> 那这个是什么标签呢，其实啊这个标签是多行文本输入框标签，然后就是这个标签的一些属性，首先是第一个属性，也就是row这个属性，这个属性的作用就是设置这个多行文本输入框显示的行而第二个属性是设置显示的列的
      • <button></button> 那这个是什么标签呢，其实啊这个标签是按钮标签，这个标签的话他的主要的作用就是创建一个按钮就
      • <input type = “” value = “” size = “” readonly = “” maxlenght = “” placeholder = “”/> 那这个是什么标签呢，其实啊这个标签是表单标签，然后就是这个标签的一些属性，首先是第一个属性，也就是type这个属性，这个属性的话他的主要的作用就是设置这个表单的类型的，那有多少种表单类型呢，如下图

  • 然后的话就是学习CSS语言
    • 在学习CSS之前我们要学了解几个概念，那是什么概念呢，其实啊也就是选择器和CSS属性还有CSS属性值
    • 那什么是选择器呢
      • 其实啊选择器也很好理解，其实就是选择html元素的方法
    • 那什么是CSS属性呢

      其实啊这个CSS属性这个概念也很好理解，其实啊CSS属性就是你要修改这个元素的具体的样式的名称，这就是在CSS中的属性

    • 那什么是CSS属性值呢

      其实啊也很简单，其实啊CSS属性值就是你要修改这个属性的具体的值，这就是在CSS中的属性值

    • 了解完这几个概念之后呢我们就可以开始学习CSS了
    • 首先呢我们先学习在CSS中有多少种常用的CSS的选择器吧
      • 首先是第一种选择器，他的名字叫做标签选择器，从名字上我们就知道他是通过标签来选择元素的，但是呢他是选择在html文件中全部的这种标签的元素，也就是说如果是通过这个选择器选择的元素的话你设置的样式将应用在这个html网页中的这种标签上，这种的选择器的格式如下：

        p{

        }

      • 然后就是第二种选择器，他的名字叫做类选择器，这个选择器的话他是选择的是在html网页中所以用class属性来设置的相同类名的元素，这个选择器的格式如下：

        .name1{

        }

        但是呢这里有一点要注意的就是一定要在类的前面加上点

      • 然后就是第三种选择器，他的名字叫做id选择器，这个选择器的话他是选择的是在html的标签中用id这个属性来设置的id的名字相同的元素，这个选择器的格式如下：

        #name1{

        }

        这里的话还要注意的一点就是一定要在id的前面加上井号

      • 然后的话就是地四种选择器，这个选择器的名字叫做后代选择器，这个选择器的话他是选择在某个元素的里面的某个元素，这个选择器的格式如下：

        div h1{

        }

      • 然后的话就是第五种选择器，这个选择器他的名字叫做组合选择器，这个选择器的话就是可以同时通过不同的选择器来选择多个元素，并且啊每个选择器之间用逗号隔开，这个选择器的格式如下：

        h1,h2,h3{

        }

      • 然后放入话就是第六种选择器，这个选择器的名字叫做伪类选择器，这个选择器的话他选择的是某个元素中的特定的状态，这个选择器的格式如下：

        a:hover{

        }

    • 学习完了在CSS中那些常用的选择器之后呢我们就可以开始学习如何修改元素的样式了，首先呢我们还要先学习一下CSS代码的格式：

      选择器{

      属性：属性值,

      属性：属性值，

      }

      如上就是我们的CSS的代码的基本的格式了，我们可以看到在CSS中每条声明都是有键值对所组成的，并且啊声明和声明之间使用逗号来隔开的

    • 之后呢学习完了CSS代码的基本格式了之后呢我们就可以开始学习在我们的html中该如何引入我们的CSS代码，其实啊在html中引入CSS代码的话有三种方法，分别是这以下几种
      • 首先是第一种方法，第一种方法的话就是通过在html中的标签的style这个属性来引入我们的CSS代码
      • 然后就是第二种引入CSS代码的方法，也就是通过style标签来引入我们的CSS代码
      • 然后就是第三种引入CSS代码的方法，也即是通过<link rel = “stylesheet” href = “”/>这个标签来引入外部的CSS的代码，那这个标签的这些属性是什么呢，首先是rel这个属性，这个属性的话他是指定我们要引入的资源，就不如说这个属性值stylesheet这个属性值的意思就是我们要引入的资源是CSS代码文件，然后的话就是href这个属性，这个属性的话就是指定这个css代码文件的路径的
    • 之后呢我们学会了如何引入我们的CSS代码了之后呢我们就可以开始学习在我们的CSS中那些各种各样的属性了
